# Creating Custom Tools

Build custom tools to extend your AI agents' capabilities with specific functionalities.

## Tool Structure

Every custom tool consists of:

- Name and description
- Input/output schema
- Implementation logic
- Error handling
- Documentation

## Basic Implementation

```typescript
// Create a basic tool
const calculatorTool = createTool({
  name: 'calculator',
  description: 'Performs mathematical calculations',
  inputSchema: {
    type: 'object',
    properties: {
      operation: { type: 'string', enum: ['add', 'subtract', 'multiply', 'divide'] },
      numbers: { type: 'array', items: { type: 'number' } }
    },
    required: ['operation', 'numbers']
  },
  async execute(input) {
    const { operation, numbers } = input;
    switch (operation) {
      case 'add':
        return numbers.reduce((a, b) => a + b, 0);
      case 'multiply':
        return numbers.reduce((a, b) => a * b, 1);
      // Add other operations
    }
  }
});
```

## Advanced Features

### Async Operations

```typescript
const asyncTool = createTool({
  name: 'dataFetcher',
  async execute(input) {
    const response = await fetch(input.url);
    return await response.json();
  }
});
```

### State Management

```typescript
const statefulTool = createTool({
  name: 'counter',
  state: { count: 0 },
  async execute(input) {
    this.state.count += input.increment;
    return this.state.count;
  }
});
```
