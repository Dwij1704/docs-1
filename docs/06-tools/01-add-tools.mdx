# Adding Tools to Agents

Tools are essential components that enable AI agents to perform specific actions and interact with external systems.

## Tool Types

xpander.ai supports several types of tools:

- System Tools
- Custom Functions
- API Integrations
- File Operations
- Database Connectors
- External Services

## Adding Tools

There are multiple ways to add tools to your agents:

1. Using the Agent Builder interface
2. Programmatically via SDK
3. Through configuration files

## Tool Configuration

// Example of adding a tool to an agent
const agent = new Agent();
