## Advanced Examples

<CardGroup cols={2}>
  <Card title="Multi-Agent System" icon="network-wired">
    ```typescript
    // Create supervisor agent
    const supervisor = await client.createAgent({
      name: 'Supervisor',
      role: 'coordinator'
    });

    // Create worker agents
    const workers = await Promise.all([
      client.createAgent({ name: 'Worker1', role: 'processor' }),
      client.createAgent({ name: 'Worker2', role: 'analyzer' })
    ]);

    // Set up communication
    await supervisor.connectTo(workers);
    ```
  </Card>

  <Card title="Voice Interface" icon="microphone">
    ```typescript
    const voiceAgent = await client.createAgent({
      name: 'VoiceAssistant',
      interfaces: ['voice']
    });

    await voiceAgent.configureVoice({
      language: 'en-US',
      voice: 'neural-1',
      speed: 1.0
    });

    voiceAgent.on('speech', async (audio) => {
      const text = await voiceAgent.transcribe(audio);
      const response = await voiceAgent.process(text);
      await voiceAgent.speak(response);
    });
    ```
  </Card>
</CardGroup>

## Integration Examples

<CardGroup cols={2}>
  <Card title="Slack Bot" icon="slack">
    ```typescript
    const slackBot = await client.createAgent({
      name: 'SlackAssistant',
      interfaces: ['slack']
    });

    await slackBot.configureSlack({
      token: process.env.SLACK_TOKEN,
      channels: ['general']
    });

    slackBot.on('message', async (event) => {
      const response = await slackBot.process(event.text);
      await slackBot.reply(event.channel, response);
    });
    ```
  </Card>

## Custom Tools

<CardGroup cols={2}>
  <Card title="Weather Tool" icon="cloud-sun">
    ```typescript
    await agent.addTool({
      name: 'weather',
      description: 'Get weather information',
      function: async ({ location, unit = 'celsius' }) => {
        const weather = await getWeatherData(location);
        return formatWeather(weather, unit);
      }
    });
    ```
  </Card>

  <Card title="File Processor" icon="file">
    ```typescript
    await agent.addTool({
      name: 'fileProcessor',
      description: 'Process different file types',
      function: async ({ file, type }) => {
        switch (type) {
          case 'pdf':
            return processPDF(file);
          case 'image':
            return processImage(file);
          default:
            throw new Error('Unsupported file type');
        }
      }
    });
    ```
