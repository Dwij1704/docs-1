Learn how to build AI agents with Recall.ai access using xpander.ai. This integration enables intelligent meeting-aware AI assistants that can access, process, summarize, and analyze meeting recordings and transcripts automatically.

    ### Let's create the Xpander.ai Recall interface!
    Write your interface name, select the interface access you'd like, Select the API key authentication method,
      select `Custom` as the Auth type, Enter `Authorization` as the Custom Header name, and paste the token copied
      earlier in the API Key field, like so:

		<Frame>
			![](/images/screenshots/recall_2025_2025-04-28-12-32-31.png)
        </Frame>

    Then save the interface.

    For more information about the Recall API authentication, [click here](https://docs.recall.ai/reference/authentication).
