## Authentication Options

Choose the authentication method that best fits your needs:

<Tabs>
  <Tab title="API Access">
    <Frame caption="Google Calendar API access configuration">
      ![Google Calendar API Access](/images/placeholders/google-calendar-api.png)
    </Frame>

    
    For programmatic access and automation:

    1. Go to [Google Cloud Console](https://console.cloud.google.com/)
    2. Create a new project or select an existing one
    3. Enable the **Google Calendar API** in Library
    4. Go to **Credentials** and create an **API Key**
    5. Restrict the API key to Google Calendar API only
    6. Copy the API key to xpander.ai's Google Calendar connection settings
    7. Set appropriate API access restrictions in Google Cloud Console
    8. Make sure that your desired Google Calendar is `available to public` by going into the specific calendar's settings, and marking the `available to public`, like so:
	
    <Frame>
		![](/images/screenshots/g_calendar_2025-04-27-14-18-21.png)
	</Frame>

    <Warning>
        API keys provide limited functionality and are best for read-only operations on public files. For full access and to keep calendars private, use OAuth methods.
    </Warning>
    
   ### Let's create the Xpander.ai Google Calendar App connection!

    Then save the interface.

   For more information about the Google API enablement, [click here](https://support.google.com/googleapi/answer/6158841?hl=en).

  </Tab>

  <Tab title="Built-in xpander.ai Auth">
    <Frame caption="Sign in with Google using xpander.ai's built-in authentication">
      ![Sign in with Google](/images/placeholders/sign-in-with-google.png)
    </Frame>

    The simplest way to connect Google Calendar is using xpander.ai's built-in authentication:

    1. Go to the **Apps** section in xpander.ai dashboard
    2. Select **Google Calendar** from available integrations
    3. Click **Sign in with Google Calendar**:
    <Frame>
		![](/images/screenshots/g_calendar_2025-04-24-14-38-57.png)
	</Frame>
    4. Select your Google account and grant permissions
    5. Your connection is immediately active and ready to use.

    <Note>
    This method is recommended for most users. xpander.ai securely manages your OAuth tokens without exposing any credentials.
    </Note>

  </Tab>
</Tabs>
