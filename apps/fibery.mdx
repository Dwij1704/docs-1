Learn how to build AI agents with Fibery access using xpander.ai. This integration enables intelligent document-aware AI assistants that can read, write, organize, and analyze pages of different projects automatically.

## Authentication Options

<Tabs>
    <Tab title="API Access">
        <Frame caption="Fibery API access configuration">
            ![Fibery API Access](/images/placeholders/fibery-api.png)
        </Frame>

        Any Admin and Member user can create API keys that allow them to access their Fibery data via the API.

        For programmatic access and automation:
        1. Login at
        [https://fibery.io/login](https://fibery.io/login)
        2. Open the Settings page.
        3. In the Personal section, click API Keys.
        4. Click `Generate API Key` to create a new token:
        <Frame>
            ![](/images/screenshots/fibery_2025-04-28-11-14-11.png)
        </Frame>
            Make sure to copy it immediately, as it will only be shown once.



### Let's create the Xpander.ai Fibery interface!
        Write your interface name, select the interface access you'd like, Select the API key authentication method,
         select `Bearer` as the Auth type, and paste the token in the API Key field, like so:
        	<Frame>
		        ![](/images/screenshots/fibery_2025-04-28-09-48-23.png)
	        </Frame>

        Now save the interface.

        For more information about the Fibery REST APIs, [click
        here](https://the.fibery.io/@public/User_Guide/Guide/Fibery-API-keys-252),
    </Tab>
</Tabs>
