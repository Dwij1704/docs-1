The `LLMTokens` model holds token counts for a single LLM call.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `completion_tokens` | `number` | Tokens generated by the model |
| `prompt_tokens` | `number` | Tokens from the user prompt |
| `total_tokens` | `number` | Sum of prompt and completion tokens |

## Example

```python Python
from xpander_sdk import LLMTokens

usage = LLMTokens(completion_tokens=123, prompt_tokens=45)
print(usage.total_tokens)
```
