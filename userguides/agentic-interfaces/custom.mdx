---
title: "Custom Agentic Interfaces"
description: "Generate your own custom interfaces to any system"
---

The more systems your AI Agent can connect to, the more useful it will be. If you need to execute function calling against a system that doesn't have a pre-made Agentic Interface, you can create your own.

<Steps>
  <Step title="Prepare required data">
    Find the required input data in order to generate your Custom interface

    <Accordion title="Supported Sources for Generating Agentic Interfaces">
      * Postman (Any collection)

      * Swagger (Any version)

      * OpenAPI Specification (Any version)

      * Public API documentation (Any accessible URL with natural language documentation)
    </Accordion>
  </Step>

  <Step title="Upload in the xpander.ai platform">
    Go to the Custom interfaces section and click the "New Custom Interface" button
  </Step>

  <Step title="Use in your AI Agent">
    In the AI Agent builder, select your new Custom Interface and start designing your AI Agent to leverage it
  </Step>
</Steps>
