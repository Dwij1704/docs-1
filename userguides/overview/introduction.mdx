# Overview

 This guide provides everything you need to know to build, deploy, and optimize powerful AI Agents using xpander.ai. Whether you’re developing task-oriented AI Agents, building connectors, or integrating complex multi-step workflows, our platform is designed to simplify the process and deliver high-value automation for your business applications.

## What is xpander.ai?

xpander.ai is an agent builder platform that enables the creation of intelligent AI Agents capable of autonomous decision-making, task execution, and adaptive behavior.

## Key Features

- Agent Graph System: xpander.ai’s agent-graph system enables developers to map complex, multi-step tasks seamlessly. Agents adapt their actions based on conditions, logic flows, and data sources, functioning as autonomous actors within your organization.
- Dynamic API Integrations: With pre-built connectors and an open integration architecture, xpander.ai allows your agents to interact with numerous platforms and custom systems. Developers can extend agent functionality with custom connectors, tailored to specific enterprise needs.
- Function Calling: xpander.ai’s robust function-calling capability enables agents to interact with APIs in precise, flexible ways, handling complex queries and transformations with ease.
- OpenAPI Compatibility: Leveraging OpenAPI, xpander.ai simplifies the creation and management of connectors, allowing for seamless agent-to-API interactions, even for multi-step or conditional workflows.

## Who is This Documentation For?

This documentation is intended for developers, data engineers, and AI teams who want to build, manage, and scale AI Agents. From creating custom connectors to fine-tuning agent workflows, xpander.ai offers a rich toolkit for integrating with diverse systems and automating complex business processes.
