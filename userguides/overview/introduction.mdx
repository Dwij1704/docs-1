## How to Get Started
<Steps>
  <Step title="Build Agent Actions using the AI Agent Builder">
    Use the [visual builder](https://app.xpander.ai) to create your agent.
  </Step>
  <Step title="Generate Agentic Interfaces to any system">
    Generate agentic interfaces or use our library of predefined agentic interfaces
    <Accordion title="Supported Sources for Generating Agentic Interfaces">
    - Postman (Any collection)
    - Swagger (Any version)
    - OpenAPI (Any version)
    - Public API documentation (Any accessible URL with natural language documentation)
    - HAR files (Any API calls made with browser)
</Accordion>
  </Step>
  <Step title="Option 1: Use xpander Agent Runtime">
    Utilize the [xpander Agent Runtime](/userguides/core-concepts/agent-runtime) for seamless integration and deployment.
  </Step>
  <Step title="Option 2: Bring Your Own LLM">
    [Bring your own LLM](/userguides/get-started/bring-your-own-llm) to use with the xpander platform.
  </Step>
</Steps>
