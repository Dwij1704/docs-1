## Getting started

<Steps>
  <Step title="Build your first AI Agent">
    Use the [AI Agent Builder](https://app.xpander.ai) to create your AI Agent.
  </Step>

  <Step title="Connect your AI Agent to any system">
    Use pre-made Agentic Interfaces or easily create your own

    <Accordion title="Supported Sources for Generating Agentic Interfaces">
      * Postman (Any collection)

      * Swagger (Any version)

      * OpenAPI (Any version)

      * Public API documentation (Any accessible URL with natural language documentation)

      * HAR files (Any API calls made with browser)
    </Accordion>
  </Step>

  <Step title="Option 1: Use xpander hosted Agent runtime">
    Utilize the as a serverless runtime - no need to write code or deploy
  </Step>

  <Step title="Option 2: build with the xpander SDK">
    Use the xpander SDK to create a supercharged AI Agent with your own code
  </Step>
</Steps>
