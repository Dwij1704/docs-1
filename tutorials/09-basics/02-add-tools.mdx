# Adding Tools to Your Agent

Learn how to extend your agent's capabilities by adding tools.

## Prerequisites

Make sure you have:
- Completed the [Create Your First Agent](/tutorials/basics/create-agent) tutorial
- xpander.ai SDK installed
- A working agent instance

## Basic Tool Structure

```typescript
interface Tool {
  name: string;
  description: string;
  function: (input: any) => Promise<any>;
}
```

## Add a Simple Calculator Tool

```typescript
await agent.addTool({
  name: 'calculator',
  description: 'Performs basic math operations',
  function: async (input) => {
    const { operation, numbers } = input;
    switch (operation) {
      case 'add':
        return numbers.reduce((a, b) => a + b, 0);
      case 'multiply':
        return numbers.reduce((a, b) => a * b, 1);
      default:
        throw new Error('Unsupported operation');
    }
  }
});
```

## Add an API Tool

```typescript
await agent.addTool({
  name: 'weatherAPI',
  description: 'Get weather information for a location',
  function: async (input) => {
    const { location } = input;
    const response = await fetch(
      `https://api.weather.com/v1/current?location=${location}`,
      {
        headers: {
          'Authorization': `Bearer ${process.env.WEATHER_API_KEY}`
        }
      }
    );
    return response.json();
  }
});
```

## Tool with Input Validation

```typescript
await agent.addTool({
  name: 'textAnalyzer',
  description: 'Analyzes text content',
  function: async (input) => {
    // Input validation
    if (!input.text || typeof input.text !== 'string') {
      throw new Error('Invalid input: text must be a string');
    }

    return {
      length: input.text.length,
      words: input.text.split(' ').length,
      sentiment: analyzeSentiment(input.text)
    };
  }
});
```

## Using Tools in Conversation

```typescript
// The agent will automatically use appropriate tools
const response = await agent.chat({
  message: 'What is 5 + 3, and what\'s the weather in London?'
});

console.log(response.content);
```

## Complete Example

```typescript
import { XpanderClient } from '@xpander/sdk';

async function main() {
  const client = new XpanderClient({
    apiKey: 'your-api-key'
  });

  const agent = await client.createAgent({
    name: 'ToolAgent',
    description: 'An agent with various tools',
    model: 'gpt-4'
  });

  // Add calculator tool
  await agent.addTool({
    name: 'calculator',
    description: 'Performs basic math operations',
    function: async (input) => {
      const { operation, numbers } = input;
      return numbers.reduce((a, b) => a + b, 0);
    }
  });

  // Add weather tool
  await agent.addTool({
    name: 'weather',
    description: 'Get weather information',
    function: async (input) => {
      const { location } = input;
      // Implementation
      return { temperature: 20, conditions: 'sunny' };
    }
  });

  // Test the tools
  const response = await agent.chat({
    message: 'What is 5 + 10, and how\'s the weather in London?'
  });

  console.log(response.content);
}

main().catch(console.error);
```
