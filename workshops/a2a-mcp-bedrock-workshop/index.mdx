Welcome to this hands-on workshop! You'll learn how to build powerful AI agent systems using xpander.ai's Agent-to-Agent (A2A) framework, Model Context Protocol (MCP), and Amazon Bedrock that seamlessly integrate with your development environment.

<Frame>

![A2A Workshop Banner](/images/workshops/Github.png)

</Frame>

## What You'll Build

In this workshop, you'll create a complete AI agent ecosystem that can:

- Collaborate on coding tasks with multiple specialized agents
- Automate code review and pull request workflows
- Orchestrate complex development processes
- Deploy production-ready agent applications

## Prerequisites

- An AWS account with Amazon Bedrock access
- A xpander.ai account
- Cursor IDE installed (with MCP support)
- Basic understanding of Python
