## üîê Step 2: Configure Environment

Copy the example environment file and populate it with your credentials from [xpander.ai Cloud](https://app.xpander.ai):

```bash
mv .env.example .env
```

Fill in the following:

```env
VITE_APP_API_KEY=your_api_key_here
VITE_APP_AGENT_ID=your_agent_id_here
VITE_APP_ORGANIZATION_ID=your_organization_id_here
```

These values are available under your account settings in the xpander.ai platform.
