## Quick Frontend Setup

### 1. Create React App
```bash
npx create-react-app swe-agent-frontend
cd swe-agent-frontend
npm install @mui/material axios react-syntax-highlighter
npm start
```

### 2. Essential Code Changes (5 lines)

**Add API client:**
```javascript
// src/api.js
const API_BASE = 'https://api.xpander.ai';
export const executeTask = (input) => fetch(`${API_BASE}/agents/YOUR_AGENT_ID/execute`, {
  method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ input })
}).then(r => r.json());
```

**Update App.js:**
```jsx
// src/App.js - Add these 5 lines to default React app
import { useState } from 'react';
import { executeTask } from './api';

// Add inside App component:
const [task, setTask] = useState(''); const [result, setResult] = useState('');
const handleSubmit = async () => { const res = await executeTask(task); setResult(res); };

// Add to JSX:
<input value={task} onChange={(e) => setTask(e.target.value)} placeholder="Describe coding task..." />
<button onClick={handleSubmit}>Execute</button> <pre>{JSON.stringify(result, null, 2)}</pre>
```
