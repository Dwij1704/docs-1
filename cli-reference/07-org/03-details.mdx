# Organization Details Command

The `org details` command displays comprehensive information about a specific organization.

## Usage

```bash
xpander org details <org_id> [--format <format>]
```

## Options

| Option | Description |
| --- | --- |
| `--format <format>` | Optional. Output format: `default` (human-readable) or `json` |

## Notes

- Use this command to:
  - Monitor resource usage
  - Track team activity
  - Review access control
  - Check subscription status
- All timestamps are in UTC
- Usage metrics are from the last 30 days
- Resource limits depend on:
  - Subscription plan
  - Custom quotas
  - Add-on features
- Access control shows:
  - Member roles
  - Authentication methods
  - Pending invitations
