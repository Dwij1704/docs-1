# Logout Command

The `logout` command ends your current session and removes stored credentials from your system.

## Usage

```bash
xpander auth logout
```

## Options

This command has no additional options.

## Examples

```bash
xpander auth logout
```

## Response

On successful logout:

```bash
âœ“ Successfully logged out
All credentials have been removed from your system
```

## Notes

- This command will:
  - End your current session
  - Remove stored credentials from your system's keychain
  - Clear any cached data
- You will need to log in again to use the CLI
- Any running processes or agents will continue to function with their existing credentials
